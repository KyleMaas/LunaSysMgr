Detailed Description

Represents a WebKit-based web page

This class seems to mostly just aggregate functionality from other classes into a single convenient point. For the most part it adds very little functionality.

Does not contain display management and rendering functionality.

See also:
WebPageClient
Todo:
There is a lot of functionality in this class that relies on other classes which are currently undocumented or not publicly documented. When some of them are, please revisit this and document it more fully.

Cleans up this WebPage

  • Lets WebAppManager know that it's closing.
  • Cleans up the view and/or the Palm::WebPage class (which is currently undocumented in this project).
  • Disconnects all child pages (pages opened from links in this page) from this one.
  • Clears all "page loading" indicators.
