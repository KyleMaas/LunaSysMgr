Detailed Description

Device-specific functionality for the Pixie devices

Device details:

  • ARMv6, dual-core.
  • No turbo mode.
  • Switches (2): ringer and headphones inserted.
See also:
https://en.wikipedia.org/wiki/Palm_Pixi

void HostArmPixie::getInitialSwitchStates ( void  ) [virtual]

Asks the HIDD service for the state of each of the switches

Posts KeyPress/KeyRelease events to the event queue of the active window for each of the hardware switches of this device.

Possible keys include:

  • Qt::Key_unknown
  • Qt::Key_Ringer
  • Qt::Key_Slider
  • Qt::Key_HeadsetMic
  • Qt::Key_Headset
