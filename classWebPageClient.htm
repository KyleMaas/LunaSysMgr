virtual void WebPageClient::invalContents ( int  x,
int  y,
int  width,
int  height 
) [pure virtual]

Invalidate a portion of the display and ask for it to be repainted

Should be called any time this window is overlapped or otherwise has a portion of it covered where it needs to be redrawn.

This method clips the rectangle to the window and adds it to the redraw list. Then, it sets a timer to redraw that portion.

Note:
Coordinates are given in window-space pixels.
Parameters:
xLeft coordinate of the rectangle to repaint.
yTop coordinate of the rectangle to repaint.
widthWidth of rectangle to repaint.
heightHeight of rectangle to repaint.
