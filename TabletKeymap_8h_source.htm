00020 /*
00021  * Tablet Keyboard - Keymaps - Layout Families and more...
00022  * --The guide--
00023  * 
00024  * This is a somewhat complete guide into how to set up a new keyboard
00025  * layout for Luna Sys Manager. Please help out and fill out the guide if
00026  * something is missing.
00027  * 
00028  * --Wordlist
00029  * Autocorrect Language - the language used by the autocorrection service. An
00030  *                        autocorrect language does not imply that the keyboard
00031  *                        language is the same.
00032  *                        autocorrect language != keyboard language
00033  * Layout Family        - a specific keyboard language, e.g. English, French
00034  * Keymap               - a keyboard layout for a layout family. One layout
00035  *                        family can have many keymaps but not the other way
00036  *                        around.
00037  * Keyboard Language    - see layout family
00038  * 
00039  * --Files and their responsibilities
00040  * TabletKeymap.cpp/.h holds the actual keyboard layouts and keyboard languages
00041  * TabletKeyboard.cpp/.h handles drawing the keyboard and interfacing with other components
00042  * VirtualKeyboard.cpp/.h parent class of TabletKeyboard, more or less just an interface
00043  * VirtualKeyboardPreferences.cpp/.h handles storage of keyboard preferences, reading them and knowing which layout to display when
00044  * 
00045  * --Creating your own keyboard layout
00046  * 1. All layout files are stored in the tabletkeymaps folder
00047  * 2. If your layout is part of a new layout family, copy any of the current
00048  *    keymap files.
00049  * 3. Open the file and edit the keymap (will not go into details about this
00050  *    step since it's quite straight forward). Make sure you always keep 12
00051  *    keys per row. Add and remove NOKEY-definitions as needed.
00052  * 4. Register your new layout family by including the file you've created
00053  *    in TabletKeymap.cpp. Make sure to add the layouts in alphabetical
00054  *    descending order, otherwise they won't appear in ascending order
00055  *    in the regional settings app.
00056  * 5. If you are using some obscure character that is not recognize yet,
00057  *    you may add it to the function UKeyIsCharacter in this file.
00058  * 5. Compile and happy typing! :-)
00059  */
